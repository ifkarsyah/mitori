# こそあど - demonstrative system (this/that/where/how)

A systematic family of reference words built on four prefixes — こ (near speaker), そ (near listener / already mentioned), あ (far from both / shared knowledge), ど (question) — each combined with the same set of suffixes for "thing," "place," "direction," "kind," and "manner." Learn the pattern once, get the whole system.

## The grid

| | こ (near me) | そ (near you / mentioned) | あ (far from both) | ど (which/where?) |
|---|---|---|---|---|
| Thing | これ (this) | それ (that) | あれ (that, over there) | どれ (which one) |
| Place | ここ (here) | そこ (there) | あそこ (over there) | どこ (where) |
| Direction | こちら/こっち (this way) | そちら/そっち (that way) | あちら/あっち (that way, far) | どちら/どっち (which way) |
| Kind (+noun) | この (this ~) | その (that ~) | あの (that ~, far) | どの (which ~) |
| Adjectival | こんな (this kind of) | そんな (that kind of) | あんな (that kind of, far) | どんな (what kind of) |
| Manner | こう (like this) | そう (like that) | ああ (like that, far) | どう (how) |

## こ/そ/あ — physical distance vs. conversational distance
Two overlapping systems:

**Physical**: こ = close to the speaker, そ = close to the listener, あ = far from both.
- これ、ください。— This one, please. (near you, the speaker)
- それ、何ですか。— What's that? (near the listener)
- あそこに猫がいる。— There's a cat over there. (far from both)

**Conversational/discourse**: そ = something the listener just mentioned or that's newly introduced into the conversation; あ = something both speaker and listener already know about/share context on (shared memory, not physically present).

- A: 昨日、いい映画を見たよ。— I saw a good movie yesterday. B: それ、何て映画? — What movie [is] that? (そ — referring to what was just said, not physically present)
- あの店、また行きたいね。— [I] want to go to that shop again, huh. (あ — a shared reference both people already know, not currently visible)

## この/その/あの/どの — before a noun (attributive)
These must be directly followed by a noun — they cannot stand alone the way これ/それ/あれ/どれ can:

- この本。— This book. (not ~~これ本~~)
- どの人ですか。— Which person is it?

Note: あの here is the same word as the filler あの (see [[../5-filler/2-あの]]) — genuinely the same demonstrative, just grammaticalized into a filler/attention-getter when used alone at the start of a sentence.

## こんな/そんな/あんな/どんな — describing a kind/type
Function like な-adjectives, describing "this kind of," "that sort of":

- こんな服が好きです。— [I] like this kind of clothes.
- そんなこと言わないで。— Don't say something like that. (see [[2-requests]] for ないで)
- どんな音楽が好きですか。— What kind of music do [you] like?

## こう/そう/ああ/どう — manner ("like this/that/how")
Describe *how* something is done or *how* something is:

- こうしてください。— Please do [it] like this.
- どうしましたか。— What happened? / What's wrong? (extremely common daily question — see the んですか pairing in [[1-んです]]: どうしたんですか)
- そう思います。— I think so. (そう here overlaps with the appearance/hearsay そう — see [[5-appearance-hearsay]])

## Notes
- どう/どんな/どちら are the question-row — they combine with か to form indefinites, mirroring 何か/誰か (see [[../1-particle/7-か]]): どこか (somewhere), どれか (some one of them).
- こちら/そちら/あちら/どちら (formal) vs こっち/そっち/あっち/どっち (casual) — same meaning, different register; the casual っち forms are far more common in daily speech.
