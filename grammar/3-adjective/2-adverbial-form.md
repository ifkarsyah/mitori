# Adverbial form — turning adjectives into adverbs

Turns "expensive/quiet" into "expensively/quietly" — modifying a verb instead of a noun. Small, mechanical rule, but used constantly in daily speech (早く来て! — come quickly!). Builds on the い/な split in [[1-i-vs-na-adjective]].

## い-adjectives: い → く
Drop the final い, add く:

- 早い → 早く — early/quickly: 早く起きた。— [I] got up early.
- 高い → 高く — expensively: 高く売れた。— [It] sold for a lot.
- おいしい → おいしく: おいしく作る。— [To] make [it] tasty.

This is the exact same い→く shift used in negative conjugation (高くない, see [[1-i-vs-na-adjective]]) — the adverbial form is just that shift without ない attached.

## な-adjectives: な → に
Drop な, add に:

- 静か(な) → 静かに — quietly: 静かに話してください。— Please speak quietly.
- 上手(な) → 上手に — skillfully: 上手に歌う。— [To] sing well.
- きれい(な) → きれいに: きれいに片付けた。— [I] cleaned [it] up nicely.

## Common daily uses

**Modifying how an action is done** (the core use):
- 元気に育つ。— [To] grow up healthy/energetic.
- 楽しく過ごしてください。— Please have a good/enjoyable time. (楽しい→楽しく)

**With なる (to become)** — extremely common pattern, "become [adjective]":
- 寒くなる。— [It]'s getting cold. (寒い→寒く+なる)
- 静かになる。— [It]'s becoming quiet. (静か+に+なる)
- 上手になりました。— [I] got good at [it].

This adjective+adverbial-form+なる pattern is one of the highest-frequency ways to talk about change in daily conversation — worth internalizing alongside the raw conjugation rule.

**With する (to make/do)** — "make [something adjective]":
- 部屋をきれいにする。— [To] make the room clean/tidy [it] up.
- 声を大きくしてください。— Please make [your] voice louder. (大きい→大きく)

Mirrors なる: なる = something becomes X on its own, する = someone actively makes something X.

## Notes
- いい's adverbial form uses the irregular よく (from 良い), not いく — 仲良く (get along well), よく食べる (eat a lot/often).
- The adverbial form of a な-adjective (Xに) looks identical to the destination/time particle に (see [[3a-に]] in 1-particle) — they're unrelated grammatically, just a coincidence of spelling; context makes it obvious which is which.
