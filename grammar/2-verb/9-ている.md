# ている — progressive vs. resultant state

て-form + いる (see [[2-て-form]]) is introduced early as "-ing," but it actually covers two different meanings depending on the verb — mixing them up is a common source of confusion in daily conversation.

## Forming it
Standard て-form attachment: 食べて + いる → 食べている. Conjugates as a る-verb from there:

- 食べている / 食べていない / 食べていた / 食べていなかった
- Polite: 食べています / 食べていません / 食べていました

Casual speech routinely drops the い: 食べてる, 食べてない — extremely common, safe to use casually, but keep the full いる form in polite/written contexts.

## Reading 1 — progressive (action in progress)
With **continuous/durative verbs** (actions that take time to complete), ている describes the action happening right now, mid-way through:

- 今、ご飯を食べている。— [I]'m eating right now.
- 雨が降っている。— It's raining. (ongoing)
- 本を読んでいます。— [I]'m reading a book.

This is the "-ing" reading most learners default to — correct for this class of verbs.

## Reading 2 — resultant state (the result of a change persists)
With **instantaneous/punctual verbs** (actions that happen in a single moment, not drawn out — 死ぬ, 結婚する, 知る, 落ちる, 開く), ている instead describes the *state that results* from that instant having already happened:

- 結婚している。— [I]'m married. (NOT "I'm in the middle of marrying" — the event already happened, and the resulting state persists)
- 知っている。— [I] know. (the moment of learning already happened; knowing is the resulting state)
- 窓が開いている。— The window is open. (someone opened it; open is the resulting state)
- 彼は死んでいる。— He is dead. (not "he is dying")

This trips learners up specifically because English "-ing" would suggest the action is mid-way, but for these verbs Japanese ている always means the state-after-the-fact, never "in the process of."

## How to tell which reading applies
Ask whether the verb describes something that takes a stretch of time (progressive reading) or something that happens in an instant, after which a new state holds (resultant-state reading). A few genuinely ambiguous verbs (着る — to wear/put on) can go either way and need context:

- 今、服を着ている。— [I]'m putting on clothes. (in progress) OR [I]'m wearing clothes. (state) — both readings exist; context/tone disambiguates.

## ている vs てある
てある is related but distinct — it marks a deliberately-arranged resultant state, usually of something someone did on purpose to an object, and implies the state was set up intentionally (often for a purpose):

- 窓が開いている。— The window is open. (neutral observation of a state)
- 窓が開けてある。— The window has been [deliberately] left open. (implies someone did it on purpose, often "in preparation for something" — see て + おく in [[2-て-form]], which is the deliberate-preparation act; てある is the resulting state of that act)

てある is lower-frequency in casual daily talk than ている — recognize it, but ている does most of the daily work.

## Notes
- 知る is a common trap: 知る itself means "to learn/find out" (a moment), so "I know" is always 知っている, never the bare plain-form 知る in the present-state sense — 知りません (I don't know) is the one common exception where the plain negative works directly without ている.
- Habitual/repeated actions can also use ている: 毎日走っている。— [I] run every day. (a third loose use, close to "I habitually do X," distinct from both readings above but easy to fold into the "ongoing state" intuition once you're comfortable with the core two.)
