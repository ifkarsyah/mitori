# Conditionals — たら / ば / と / なら

Japanese has four conditional forms where English just has "if." They overlap heavily but aren't interchangeable. Per [[0-overview]], たら is the one to master first — it's the most flexible and covers casual daily "if/when" almost entirely; treat ば/と/なら as later refinements, not day-one material.

## たら — the default, flexible conditional
Forms off plain past (た-form, see [[2-て-form]] for the sound-shift table) + ら:

- 食べたら → if/when [you] eat
- 高かったら → if [it]'s expensive (い-adjective past + ら)
- 暇だったら → if [I]'m free (な-adjective/noun past + ら)

たら covers one-off hypotheticals, sequences, and suggestions/requests after the fact — this breadth is exactly why it's the safe daily default:

- 雨が降ったら、行きません。— If it rains, [I] won't go.
- 駅に着いたら、電話してください。— When [you] arrive at the station, please call. (sequence: arrive, then call)
- 疲れたら、休んでください。— If/when [you] get tired, please rest.

## と — mechanical/automatic "whenever"
Already covered from the particle side in [[5a-と]] — と-conditional describes something that *always* follows automatically, not a one-off choice. Can't be used for suggestions, requests, or anything involving deliberate will in the result clause:

- 春になると、暖かくなる。— When spring comes, it gets warm. (natural law, always true)
- ~~このボタンを押すと、開けてください。~~ — ungrammatical: と can't end in a request.

## ば — formal/written, "if" with emphasis on the condition itself
Formed by changing the final -u of the plain non-past to -eba (う-verbs) or dropping る and adding れば (る-verbs):

- 読む → 読めば — if [you] read
- 食べる → 食べれば — if [you] eat
- 高い → 高ければ — if [it]'s expensive

ば feels more formal/written than たら, and is common in set phrases and proverbs:

- 聞けば分かります。— If [you] ask, [you]'ll understand.
- すみませんが、できればお願いします。— Sorry, if possible, please [do it]. (できれば is a very common fixed phrase — "if possible")

Like と, ば traditionally resists ending in a direct request/command in the strictest grammar, though this is loosened in casual modern use — たら remains the safer choice when in doubt.

## なら — conditional on a premise/topic, not an event
なら reacts to something already said or assumed — "if that's the case, then..." — and can also mean "if you're going to do X" (advice framing). It attaches to plain form (dropping the copula for nouns/な-adjectives):

- 行くなら、傘を持って行って。— If you're going [to go], take an umbrella. (reacting to their stated plan)
- 田中さんなら、知っていると思います。— If it's [about] Tanaka, I think [he]'d know. (premise: talking about Tanaka specifically)

なら is distinct from the other three because it's not really about a hypothetical event unfolding — it's about responding to a premise on the table.

## Quick comparison

| Form | Best for | Can end in request/command? |
|---|---|---|
| たら | Default, flexible, sequences | Yes |
| と | Automatic/mechanical results | No |
| ば | Formal, written, set phrases | Traditionally no (loosening) |
| なら | Reacting to a stated premise/plan | Yes |

## Notes
- All four can often be swapped for たら without sounding wrong in casual speech — the finer distinctions matter more for natural-sounding formal/written Japanese than for getting understood.
- Don't drill ば/と/なら hard until たら is automatic — per [[0-overview]], this is explicitly long-tail refinement once the workhorse form is solid.
