# たい-form — want to

Turns a verb into "want to [verb]." Built directly from the ます-stem (see [[1-conjugation-group]]), and conjugates like an い-adjective once formed.

## Forming it
ます-stem + たい, for any group:

- 食べる → 食べたい — want to eat
- 飲む → 飲みたい — want to drink
- 行く → 行きたい — want to go
- する → したい — want to do
- 来る (kuru) → 来たい (きたい) — want to come

## Conjugates as an い-adjective, not a verb
Once you've formed X-たい, treat it exactly like an い-adjective (see the adjective notes) — not like a verb:

| | Plain | Polite |
|---|---|---|
| Non-past affirmative | 食べたい | 食べたいです |
| Non-past negative | 食べたくない | 食べたくないです |
| Past affirmative | 食べたかった | 食べたかったです |
| Past negative | 食べたくなかった | 食べたくなかったです |

## Object marking: が vs を
With たい, the object can take either が or を — が leans slightly more natural/common in casual speech, を is common when emphasizing the object or in more neutral/written style:

- 水が飲みたい。/ 水を飲みたい。— [I] want to drink water.

This is the same pattern behind other feeling/desire predicates taking が (see [[1b-が]]) — たい softens but doesn't eliminate that pull toward が.

## First person (and why 3rd person is different)
Plain たい directly describes the *speaker's* own desire — using it to state someone else's desire sounds off, similar to how you can't directly say "he is sad" the same way you say "I am sad" in Japanese:

- 私は行きたいです。— I want to go. (natural)
- ~~彼は行きたいです。~~ — sounds like you're claiming direct access to his inner state.

For third parties, hedge or use different constructions — 彼は行きたがっている (he seems to want to go, たがる form) or 彼は行きたいと言っています (he says he wants to go). These are long-tail refinements; not needed to start using たい for yourself.

## Notes
- たい + と思う stacks constantly in natural speech: 食べたいと思います。— I think I want to eat. (softens the directness of stating a desire, especially in polite contexts)
- Don't confuse verb-stem+たい (want to do) with the unrelated い-adjective 高い/暑い etc. — the たい here is a suffix, not a coincidence of spelling.
